	給我你的email或是暱稱

	我是用github desktop
	下載完開啟 並 登入自己的帳號
	左上角file→clone repo 輸入這段gh repo clone robotfly112vr/robotfly_unity
	*記得選要存放的位置
-----
	使用方法：
	fetch origin：檢查是否有更新的檔案。
	pull：將檔案拉到自己的電腦上。
	push：將你自己做的檔案丟上去。
	merge：把自己的新檔案和舊有的檔案合併。
	branch：開一個分支，這個分支可以拉出來不同的功能，讓檔案不會相衝，到最後需要再和主幹合併起來。
